## シナリオ概要

依頼者・木村美穂（妻）から不貞を理由とする離婚の依頼を受任。婚姻費用月18万円・養育費月10万円・
慰謝料300万円・自宅売却差額折半・年金分割0.5・親権希望を相手方代理人に通知したところ、
第1回回答では離婚・年金分割・親権には同意しつつ金額面で大きな差（婚姻費用8万円・養育費6万円・
慰謝料100万円、自宅売却拒否）が示された。反論書面で根拠開示を求めたところ、第2回回答で
譲歩案（婚姻費用12万円・養育費8万円・慰謝料150万円・自宅売却差額折半に同意）が提示され、
依頼者の了承を得て示談成立に至った、という想定の交渉フローです。

## このテストで見つかり修正した不具合

- 見出しと本文が改行で分かれた書面形式（「第3　養育費について」を見出し行、キーワードを含む文を
  次行に記載するパターン）で、相手方回答の自動タグ付けが本文を読み取れない不具合
- 全角スペースで姓・名が区切られた氏名（例：「木村　美穂」）が依頼者・相手方とも姓のみに
  短縮されてしまう不具合
- 「異議ありません／異議ございません」という同意表現が同意タグとして認識されない不具合

なお、相手方の譲歩案が当初請求額と異なる金額で「応じる」旨を述べているケースは、自動タグでは
そのまま「同意」と判定されます（金額の差異までは自動判定せず、弁護士による内容確認・補正を
前提とした仕様としています）。
